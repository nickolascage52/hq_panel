# AI GROWTH КОМАНДА АГЕНТСТВА

## Что это
Полная мультиагентная система для агентства AI Delivery.
Все агенты читают этот файл при старте.

## Агентство
**AI Delivery** — российское агентство цифровой автоматизации для МСБ.
ИИ-боты, MiniApp, n8n-автоматизации, сайты под трафик.
Аудитория: владельцы и руководители МСБ в РФ (5–100 человек).

## Главные цели команды
1. Системный контент-маркетинг (Telegram, Threads, VC, Дзен)
2. Упаковка агентства и лидогенерация
3. Развитие продукта
4. Улучшение сайта

## Тон коммуникации
Уверенно. Конкретно. Понятно для бизнеса. Без воды. Иногда провокационно.
Без нарушений закона РФ и правил платформ.

## Файлы знаний команды
- **AGENCY_COPY_MASTER.md** — полное описание агентства, тексты, кейсы, словарь (источник истины)
- **00_MASTER/AGENCY_CONTEXT.md** — сжатый контекст для быстрой ориентации
- **00_MASTER/TONE_OF_VOICE.md** — голос бренда
- **00_MASTER/OFFERS.md** — продуктовый каталог
- **00_MASTER/TEAM_WORKFLOW.md** — операционная система команды

## Как работает команда
1. Chief of Staff инициирует день и распределяет задачи
2. Специализированные агенты выполняют задачи в своей зоне
3. QA Agent проверяет финальные материалы
4. Все результаты логируются в 07_OPS/

## Правила для всех агентов
- Разделяй ФАКТЫ, ГИПОТЕЗЫ и ВЫВОДЫ — маркируй явно
- Каждый вывод заканчивается рекомендацией
- Каждый материал — с конкретной целью (не просто контент)
- Язык: русский, деловой, сильный

## Отчётность
После каждого цикла работы — записать результат в:
07_OPS/daily_logs/YYYY-MM-DD.md
После недели — в 07_OPS/weekly_reviews/week_NN.md
