## Ключевые термины

| Термин | Значение |
|--------|----------|
| MVP | Минимально жизнеспособный продукт — первая рабочая версия (1 сценарий за 24 часа) |
| Боевой запуск | Полноценный продукт с интеграциями, срок 3–10 дней |
| Бриф | Первичное описание задачи клиента (15–30 минут) |
| MiniApp | Веб-приложение внутри Telegram |
| n8n | Инструмент визуальной автоматизации процессов с открытым исходным кодом |
| Лид | Входящая заявка от потенциального клиента |
| Pipeline / пайплайн | Цепочка обработки: лид → CRM → задачи → отчёты |
| Handoff | Передача проекта клиенту: доступы, инструкции, документация |
| Квалификация | Определение качества/приоритета лида вопросами или логикой |
| SLA | Service Level Agreement — согласованные сроки ответа/обработки |
| КП | Коммерческое предложение |
| UTM | Метки для отслеживания источника трафика в аналитике |

## Словарь бренда (использовать)

| Правильно | Вместо |
|-----------|--------|
| автоматизация | диджитализация, цифровизация |
| внедрение | имплементация, деплой |
| заявка, лид | конверсия, трафик (без объяснения) |
| клиент | юзер, пользователь (в текстах для бизнеса) |
| результат | аутпут, deliverable |
| процесс | флоу, пайплайн |
| Telegram-бот | телеграм бот (без дефиса) |
| MiniApp | мини-апп, миниапп |
| запуск | релиз, деплой |

## Избегать

- Технарщина без объяснения: webhook, endpoint, microservices
- Инфоцыганство: трансформация, прокачать, выйти на новый уровень
- Пустые обещания: гарантируем 100%, увеличим в 10 раз
- Канцелярит: в целях оптимизации, в рамках сотрудничества
