# Кейсы AI Delivery

Реальные кейсы для контента и сайта. Источник: AGENCY_COPY_MASTER.md.

## Формат кейса

```markdown
# Кейс: [Название / Ниша клиента]
Дата: YYYY-MM-DD
Ниша: [отрасль]
Размер клиента: [сотрудников / выручка]

## Ситуация
[Что было до нас — проблема, боль, цифры потерь]

## Задача
[Что нужно было решить]

## Решение
[Что мы сделали — конкретно, по шагам]

## Результат
[Цифры, метрики, эффект]
- Метрика 1: было X → стало Y
- Метрика 2: ...

## Сроки
[Сколько заняло внедрение]

## Связь с оффером
[Какую услугу иллюстрирует]
```

## Использование
- Content Strategist — для тем постов и статей
- Web Copywriter — для страниц услуг и лендингов
- Telegram Lead, VC Writer — для материалов с социальным доказательством
