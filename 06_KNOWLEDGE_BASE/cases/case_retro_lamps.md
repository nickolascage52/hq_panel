# Кейс: Сайт по продаже настольных светильников в ретро-стиле

**Ниша:** E-commerce  
**URL кейса:** /cases/retro-lamps-website

## Ситуация
Нужно увеличить продажи и средний чек через UX.

## Задача
Конвертящий сайт с брендовым интерфейсом.

## Решение
Брендовый интерфейс, карточки товаров, микроанимации, быстрый путь к покупке.
Технологии: UI Motion, Vanilla JS, Catalog UX, Analytics, Checkout Flow.

## Результат
- Конверсия 3.4%
- Средний чек +17%
- Добавления в корзину +29%

## Связь с оффером
Сайты и лендинги под трафик.
