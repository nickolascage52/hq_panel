# Кейс: Многостраничный сайт для компании по клинингу

**Ниша:** Клининг / Услуги  
**URL кейса:** /cases/cleaning-company-website

## Ситуация
Нужен конвертящий сайт с заявкой и доверием (отзывы, оффер).

## Задача
Сайт, который собирает заявки.

## Решение
Пошаговая форма, блок спецпредложений с таймером, блок отзывов с агрегаторов.
Технологии: HTML/CSS/JS, многошаговая форма, таймер акций, блок отзывов, адаптив.

## Результат
Единая точка входа для заявок, рост конверсии.

## Связь с оффером
Сайты и лендинги под трафик.
