# Кейс: Автоматизация продаж для B2B-сервиса

**Ниша:** B2B сервис  
**URL кейса:** /cases/b2b-sales-automation

## Ситуация
Менеджеры тратили время на рутину по статусам и коммерческим предложениям.
Задержки с ответами клиентам.

## Задача
Ускорить менеджеров, убрать рутину по статусам и КП.

## Решение
Pipeline с квалификацией лидов, автопроставлением статусов, автоматической генерацией КП.
Технологии: Telegram Bot, CRM, n8n, Webhooks, Google Docs API.

## Результат
- Экономия 18 ч/нед
- Скорость ответа 20 мин
- 100% статусов проставлено автоматически
- +24% обработанных лидов

## Связь с оффером
ИИ-боты «Лиды и поддержка» — квалификация + авто-КП + передача в CRM.
