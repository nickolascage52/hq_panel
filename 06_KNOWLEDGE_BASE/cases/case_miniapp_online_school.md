# Кейс: MiniApp-MVP каталога для онлайн-школы

**Ниша:** Онлайн-образование  
**URL кейса:** /cases/miniapp-mvp-online-school

## Ситуация
Клиент хотел быстро проверить интерес к новому формату без разработки отдельного сайта.
Telegram — основной канал коммуникации с аудиторией.

## Задача
MVP для проверки гипотезы: каталог + заявка внутри Telegram.

## Решение
MiniApp с каталогом и формой заявки в Telegram.
UTM-аналитика, маршрутизация лидов.
Технологии: Telegram Mini Apps, JavaScript, n8n, Google Sheets, UTM Analytics.

## Результат
- 212 заявок
- Конверсия 6.2%
- CPL −28%
- MVP за 4 дня

## Связь с оффером
Telegram MiniApp MVP — проверка спроса за дни, не недели.
