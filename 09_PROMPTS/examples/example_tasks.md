## Chief of Staff — примеры запросов

### Запуск дня
```
Прочитай CLAUDE.md, 00_MASTER/AGENCY_CONTEXT.md и 07_OPS/backlog/backlog.md.
Сегодня [ДАТА]. Сформируй план дня.
Определи топ-3 задачи, нужных агентов и ожидаемые артефакты.
Запиши в 07_OPS/daily_logs/[ДАТА].md
```

### Недельное планирование
```
Прочитай 00_MASTER/TEAM_WORKFLOW.md и текущий backlog.
Сформируй приоритеты на неделю [N]:
- Контент (что и когда)
- Исследования
- Сайт
- Продукт
Запиши план в 07_OPS/weekly_reviews/week_[N].md
```

### Генерация отчёта
```
Проверь все папки проекта и подведи итог рабочего цикла.
Создай REPORT.md с полной статистикой, артефактами и рекомендациями.
```

## AGENT TEAMS — примеры промптов для параллельной работы

> **Запуск через WSL + tmux + Claude:** см. 09_PROMPTS/LAUNCH_AGENT_TEAM.md и WSL_LAUNCH_INSTRUCTION.md

### Недельный контент-спринт
```
Создай команду для недельного контент-спринта.

Участники и задачи:
- Content Strategist: создай контент-план на неделю (сохрани в 02_CONTENT/)
- Telegram Lead: напиши 5 постов по плану (сохрани в 02_CONTENT/telegram/drafts/)
- Threads Creator: напиши 5 тредов по плану (сохрани в 02_CONTENT/threads/drafts/)
- QA Agent: проверь все созданные материалы, дай заключение по каждому

Все читают: CLAUDE.md, 00_MASTER/AGENCY_CONTEXT.md, 00_MASTER/TONE_OF_VOICE.md
После завершения — Chief of Staff генерирует REPORT.md
```

### Исследовательский спринт
```
Создай команду для исследовательского спринта.

Участники и задачи:
- Market Researcher: исследуй топ-3 боли МСБ в [СФЕРА]
- Competitor Analyst: проанализируй 3 конкурентов из [СПИСОК]
- Product Manager: на основе исследований обнови гипотезы в 04_PRODUCT/hypotheses.md
- Content Strategist: создай 10 тем для контента на основе исследований

После завершения — Chief of Staff: сводный REPORT.md
```
